1. Float:Float é uma propriedade CSS que usamos para posicionar algo na tela, seja um elemento em bloco ou em linha.
a) None:É uma palavra-chave que indica que o elemento deve flutuar em baixo do bloco.
b) Left: É uma palavra-chave que indica que o elemento deve flutuar à esquerda do bloco.
c) Right: É uma palavra-chave que indica que o elemento deve flutuar à direita do bloco.

2. Position: A propriedade controla um elemento é posicionado. existem 5 valores:
- Padrão: Todo elemento começa como estática.
a) Relative: Move o elemento em relação (da forma que deseja) a sua posição original. O espaço original é preservado. Para mover o elemento tem que ser atribuída as coordenadas(top, bottom, left, right.)
- ordem dos números no Sentido horário da contagem: Cima, direita, baixo, esquerda
b) Absoluto: Remove o elemento do fluxo. Posiciona em relação ao ancestral posicionado mais próximo (desde que não seja posição estática)
c) Fixed: Remove o fluxo e fixa na viewport(a janela do navegador). não se move ao rolar a página.
d) Sticky: Hibrido entre relative e fixed. Age como relative até atingir o ponto definido - então gruda como fixed, mas apenas dentro do seu conteiner(cabeçalho) pai.

Clearfix: é uma técnica CSS essencial usada para garantir que um elemento pai envolva totalmente seus filhos flutuantes (float), impedindo que o layout "quebre" ou colapse.

Margin: define o espaçamento externo ao redor de um elemento, fora de qualquer borda
- ordem dos números no Sentido horário da contagem: Cima, direita, baixo, esquerda
margin-bottom: define a margem área na parte inferior de um elemento. Um valor positivo coloca o valor mais longe de outros elementos vizinhos, enquanto um valor negativo, aproxima.

Border -radius: Arredonda as bordas do elemento
- ordem dos números no Sentido horário da contagem: Cima, direita, baixo, esquerda

Padding:A propriedade padding define uma a distância entre o conteúdo de um elemento e suas bordas. É um atalho que evita definir uma distância para cada lado separadamente

a) padding-left: é um elemento que define o espaço de preenchimento obrigatório no lado esquerda de um elemento.
b) padding-Right: é um elemento que define o espaço de preenchimento obrigatório no lado direita de um elemento
c) padding -top: é um elemento que define o espaço de preenchimento obrigatório em relação a parte superior de um elemento.
d) padding - bottom:é um elemento que define o espaço de preenchimento obrigatório em relação a parte inferior de um elemento.

Background: É a configuração do estilo de fundo.
Overflow-y:scroll (Caso os elementos sejam muitos grandes na tela, será criado uma barra de rolamento.
height: determina a altura da área do conteúdo de um elemento
font-weight: especifica o peso ou a intensidade da fonte
Hover: A ação de algo ao passar o mouse sobre o elemento.
Transition: É a mudança de cor ao passar o mouse sobre elemento
Transform: scale(1.2) Mudança de tamanho no icone selecionado pelo mouse
