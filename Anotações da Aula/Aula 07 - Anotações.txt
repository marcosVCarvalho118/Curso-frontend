nav = define uma seção de links de navegação, sendo fundamental para semântica, acessibilidade e SEO. Ela é usada para blocos principais de links, como menus, índices ou breadcrumbs.

Barra de navegação:
navbar = barra de navegação de celular
navbar-expand-lg = barra de navegação de computador
navbar brand = Ícone vai para direita na barra de navegação
Navbar-dark: A cor da barra de navegação será preta.
<div class="container">: é um elemento não semântico usado para agrupar outros elementos para fins de estilo ou layout.
<button class="navbar-toggler"> é um componente do Bootstrap usado para criar um botão de menu "hambúrguer" que aciona uma barra de navegação recolhível em telas menores. Ele funciona com data-bs-toggle="collapse" e data-bs-target para mostrar ou ocultar links do menu, geralmente usando <span class="navbar-toggler-icon"> para aparecer um ícone ao recolher a barra de navegação.

Espaçamento de barra de navegação:
"(<a>)" para que se pareça com um botão atribuindo a ela a classe "button"
"gap-n°": define o espaçamento (gutters) entre linhas e colunas em contêineres
"ms-auto" no Bootstrap 5 é uma utilidade de espaçamento (margin-start: auto) usada para alinhar elementos à direita dentro de um contêiner flexbox.

<section> representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo..
py-5:. É uma forma rápida de adicionar espaçamento vertical responsivo.
mb-3 no Bootstrap é uma utilidade CSS que define a margem inferior (margin-bottom) de um elemento.
display é utilizada para indicar a forma como os elementos HTML serão dispostos na página ou em espaços delimitados. Por essa característica, é um recurso essencial para manter a organização dos elementos na página.
Organização de seção e cartão:
<span class> é um conteiner generico em linha para conteúdo fraseado , que não representa nada por natureza.
<i class> : Inserir ícones no texto
"h-100" no Bootstrap define a altura (height) de um elemento como 100% em relação ao seu elemento pai
"d-flex" é um utilitário de CSS que aplica display: flex a um elemento, transformando-o em um contêiner flexbox. Isso alinha automaticamente os elementos filhos (flex items) lado a lado.
flex-direction (column): você pode organizar itens em uma coluna definindo a propriedade. Isso altera o eixo principal de horizontal (padrão) para vertical.
"card-body"  é a seção principal de um card (contêiner flexível) utilizada para adicionar espaçamento interno (padding) ao conteúdo, como textos e títulos.
 <a href=#> é usada para criar hiperlinks que conectam uma página a outra.
<footer class>, situa-se no final da página ou seção, contendo informações de direitos autorais, links de navegação, contatos e redes sociais.
